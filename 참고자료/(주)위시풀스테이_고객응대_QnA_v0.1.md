## 1. 예약 관련

**Q. 어떻게 예약하나요?**
홈페이지 또는 지정 예약 채널을 통해 예약하실 수 있습니다. 희망 날짜, 인원, 연락처를 입력해 주시면 됩니다. 결제 완료 시 예약이 확정됩니다.

**Q. 예약 가능한 채널은 어디인가요?**

## 4. 취소 및 환불

**Q. 예약을 취소하고 싶어요. 어떻게 하나요?**
예약 취소는 예약자 본인이 직접 신청해 주셔야 합니다. 예약하신 채널 또는 고객 문의 채널로 연락 주세요.

**Q. 환불 기준이 어떻게 되나요?**
체크인일 기준으로 환불 기준이 다릅니다:
- 10일 전 취소: 100% 환불
- 9~6일 전 취소: 70% 환불
- 5~2일 전 취소: 50% 환불
- 1일 전 또는 당일 취소: 환불 불가 (0%)
※ 환불 기준 시각은 자정(00:00) 기준입니다.

**Q. 취소하면 환불은 언제 되나요?**
환불은 원래 결제하신 수단으로 처리되며, 영업일 기준 3~5일 이내에 완료됩니다.

**Q. 항공편이 결항돼서 못 가게 됐어요. 환불 받을 수 있나요?**
항공편·선박 결항 등 불가항력적 사유는 관련 증빙(결항 확인서 등)을 제출하시면 특별 취소 처리가 가능합니다. 담당자에게 문의해 주세요.

**Q. 몸이 아파서 못 가게 됐는데 환불 되나요?**
개인 사유(질병, 입원 포함)는 원칙적으로 무료 취소 대상이 아닙니다. 위에 안내드린 일반 환불 기준이 적용됩니다.

**Q. 예약 날짜를 변경하고 싶어요.**
날짜 변경은 기존 예약 취소 후 재예약으로 진행됩니다. 이때 취소 시점의 환불 규정이 동일하게 적용됩니다. 담당자 확인 후 처리해 드립니다.

**Q. 외부 플랫폼(야놀자, 에어비앤비 등)으로 예약했는데 취소하려면 어떻게 하나요?**
외부 플랫폼을 통해 예약하신 경우, 해당 플랫폼의 취소·환불 정책이 우선 적용됩니다. 플랫폼 앱 또는 고객센터를 통해 취소해 주세요.

**Q. 노쇼(No-show)의 경우 환불이 되나요?**
별도 연락 없이 체크인을 하지 않으신 경우(노쇼)는 당일 취소와 동일하게 환불이 되지 않습니다.

## 10. 자주 헷갈리는 상황 모음

**Q. 예약은 했는데 결제를 아직 안 했어요.**
결제가 완료되지 않으면 예약이 자동 취소됩니다. 빠르게 결제를 완료해 주세요. 도움이 필요하시면 문의 주세요.

**Q. 예약자와 이용자가 달라도 되나요?**
[직접 입력 필요 - 예약자·이용자 불일치 허용 여부]

**Q. 이름이 잘못 입력됐어요.**
담당자에게 문의 주시면 확인 후 수정 도움드리겠습니다.

**Q. 예약 내역을 확인하고 싶어요.**
예약 시 받으신 확인 문자 또는 이메일을 확인해 주세요. 찾기 어려우시면 연락 주세요.

**Q. 체크인 안내 문자를 못 받았어요.**
체크인 당일 발송됩니다. 문자를 받지 못하셨다면 운영 시간 내 문의해 주세요. 연락처: [직접 입력 필요]

**Q. 예약이 됐는지 모르겠어요.**
결제 완료 후 확인 안내가 발송됩니다. 안내를 못 받으셨거나 불확실하시면 담당자에게 연락해 주세요.

**Q. 체크아웃 시간을 넘겼어요.**
사전 협의 없이 체크아웃 시간을 초과하신 경우 추가 요금이 발생할 수 있습니다. 즉시 담당자에게 연락해 주세요.

## 11. 챗봇 한계 안내용 문구

> 아래 문구들은 챗봇이 답변할 수 없을 때 사용하는 기본 문구입니다.

- 일반 문의 처리 불가: "해당 내용은 담당자가 직접 확인 후 안내드리는 게 정확할 것 같습니다. 빠르게 연결해 드릴게요! 조금만 기다려 주세요 :)"
- 컴플레인 접수: "불편을 드려서 정말 죄송합니다. 말씀하신 내용 바로 담당자에게 전달하겠습니다. 빠르게 확인 후 연락드리겠습니다."
- 운영 시간 외 안내: "현재 운영 시간(09:00~21:00)이 아닙니다. 문의 내용은 접수되었으며, 운영 시간 내에 순차적으로 답변드리겠습니다."
- 문서에 없는 내용: "해당 내용은 제가 바로 확인하기 어렵습니다. 담당자 확인 후 안내드리겠습니다."
